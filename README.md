# Web Studio Modern

Modern responsive landing page for a digital studio agency built with HTML, CSS, and vanilla JavaScript.

## Features

- Responsive layout for mobile, tablet, and desktop
- Sticky header with smooth scrolling navigation
- Mobile menu interaction
- Modal window for contact form
- Modern styling with CSS transitions and hover effects

## Project Structure

- `index.html` — main page markup
- `css/style.css` — styles
- `js/main.js` — entry script
- `js/mobile-menu.js` — mobile menu behavior
- `js/modal.js` — modal window behavior
- `images/` — project images and icons

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES modules)
- Modern Normalize
- Google Fonts

## Run Locally

1. Clone the repository:
   ```bash
   git clone <your-repository-url>
   ```
2. Open `index.html` in your browser, or use a live server extension in VS Code.
